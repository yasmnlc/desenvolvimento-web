Bem-vindo!

A Maplink é uma empresa multinacional de tecnologia com mais de 25 anos de experiência no mercado de tecnologia e inovação. Ajudamos empresas de todos os setores a expandirem seus negócios por meio da tecnologia e inteligência de dados de localização.

Desenvolvemos APIs de geolocalização com as melhores tecnologias do mercado, que nos permitem entregar uma solução pronta e completa para ser integrada a um sistema ou plataforma. O nosso propósito é ajudar as pessoas a encontrarem caminhos para novos lugares e experiências.

Nossas APIs

Planning

A Planning API é uma tecnologia de planejamento logístico que simplifica itinerários de operações complexas, apresentando o melhor sequenciamento de entregas para cada veículo, e com base nas necessidades do negócio. Com a Planning API você cria rotas com até 200 pontos de parada, aproveitando os trajetos para otimizar o seu transporte, ganhar mais agilidade e economizar combustível.

Toll

DA Toll API retorna detalhes completos sobre as praças de pedágio presentes na rota pré-definida: - Nome - Endereço - Valor do pedágio - Valor acumulado por trechos e para a rota completa - Concessionária responsável - Cálculo de pedágio proporcional - Coordenadas (lat/long) do pedágio - ServiceId Via Fácil – Sem Parar para integração com sistemas de pagamento vale-pedágio - ServiceId ANTT – código de identificação da ANTT (Agência Nacional de Transportes Terrestres)
